I've written about this before but I have some javascript that you can include on a page an it will go and modify all the links so that if they are off-site links they'll open in a new window (or tab if you have your browser configured right). It also hides your links behind a referrer stripping redirector. I used to use google's redirector but they shut it down so now I use the excellent redirector from hidebehind.com.

Here's fixlinks.js. I saved it as .txt to try and avoid people using it from my site on their own site. Please copy this to your own server to use it.

It adds an onload handler (and preserves any that are already there) and it also modifies every link to have an onclick handler, it also preserves exisiting onclick handlers.
