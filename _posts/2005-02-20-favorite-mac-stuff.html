  • Firefox the best web browser (see my blog about my favorite extensions)
  • Quicksilver better than spotlight, faster easier to use, and has advanced features I've not even got into using yet
  • Adium multi protocol IM client Used to crash a lot, works great now
  • Emacs native version doesn't need to run in a terminal
  • Google Stuff
    • Gmail Notifier
    • Picasaweb iPhoto Uploader
  • Skype
  • Fink Open source software package manager
  • Gimp Image Editor
  • xv Image viewer
  • SnapNDrag easy screen captures
  • VLC video player
  • MagiCal a nice tidy calendar in your menu bar
Stuff I've stopped using but you might find it useful:
  • ssh-keychain Keep SSH keys in your Apple Keychain
  • Salling clicker do amazing things on your mac via your bluetooth phone
  • Synergy Share keyboard and mouse accross platforms
  • iPhoto to gallery
  • iScroll2 tell your mac to scroll with two fingers
